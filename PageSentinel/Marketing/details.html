デモページを開く

デモページのAutoPilot の設定例

1. ボタン1 の CSS セレクタ
最初にクリックする要素を指定します。#fetchBtn
2. ボタン1 の待機時間 (秒)
クリック後に次のステップまで待機する時間を設定します。1.0
3. ボタン2 の有効/無効
2 つ目の自動クリックを使用するチェックを入れます。
4. ボタン2 の CSS セレクタ
2 つ目にクリックする要素を指定します。 #yesBtn
5. ボタン2 の待機時間 (秒)
2 回目のクリック後にサーバ応答を待つ時間を設定します。 1.0
6. 判定文字列
繰り返し続行か終了かを判定するため、継続する場合のキーワードを指定します。最新データを取得しました。
7. クローズボタンの CSS セレクタ
判定文字列が見つかった場合に閉じる要素を指定します。#okBtn
8. クローズ後の再待機時間 (秒)
判定後に再試行まで待機する時間を設定します。3.0
