1. 収集する情報

  1. ユーザー入力データ：監視対象URL、リロード間隔、CSSセレクタ等、アプリ内で入力・設定した情報。
  2. アプリ内設定：アンチフリッカー、スクロール遅延、検知モード等のUI/UX関連設定。
  3. 端末メタデータ：OSバージョン、デバイス種別（iPhone/iPad）など動作最適化に必要な技術情報。個人を特定できるID（UUID等）は取得しません。

2. 利用目的

  • 指定Webページの監視・自動リロードの実行
  • チラつき防止や自動操作（オートパイロット）等の制御
  • 設定の永続化・復元
  • 匿名化した技術情報の分析による品質改善

3. 第三者提供の有無

法令に基づく場合を除き、第三者への提供は行いません。

4. Cookie 等の取扱い

アプリはWebViewでページを表示します。CookieやlocalStorageの利用は表示サイトのポリシーに従います。アプリ側で収集・送信は行いません。

5. 保存期間

ユーザー入力および設定は端末内に保存され、アンインストールにより削除されます。外部サーバーには保存しません。

6. データセキュリティ

  • 通常、他アプリからアクセスできないローカル領域に保存します。
  • アップデート時のデータ破損を避けるための移行処理を実装しています。

7. ユーザーの権利

  • アプリ内で設定の変更・削除が可能
  • アンインストールにより全データを削除可能

8. 免責

本アプリは現状有姿で提供されます。対象サイトの構造変更等により機能へ影響が出る場合があります。

9. 方針の変更

必要に応じて本方針を変更します。変更はアプリ内または公式サイトで告知後に効力を生じます。

1. Information Collected

  1. User Input Data: URLs to monitor, reload intervals, CSS selectors, etc. entered in the app.
  2. In-App Settings: Anti-Flicker level, scroll delays, detection mode and other UI/UX settings.
  3. Device Metadata: OS version and device type (iPhone/iPad) for operation/optimization. No personally identifying IDs (e.g., UUID) are collected.

2. Purpose of Use

  • Monitoring and auto-reloading of specified web pages
  • Controls for anti-flicker and automation (Autopilot)
  • Persisting and restoring settings
  • Quality improvement via analysis of anonymized technical data

3. Third-Party Provision

No third-party sharing is performed unless required by law.

4. Cookies & Web Technologies

The app uses WebView. Any cookies/localStorage are handled by the displayed website’s policy; the app does not collect or transmit them.

5. Retention

Data is stored locally and removed upon uninstallation. No external servers are used.

6. Data Security

  • Data is kept in local storage not ordinarily accessible by other apps.
  • Migration safeguards help prevent corruption during updates.

7. User Rights

  • Modify or delete settings within the app
  • Delete all data by uninstalling the app

8. Disclaimer

The app is provided “as is.” Changes to target websites may affect functionality.

9. Changes

We may update this Policy. Updates take effect once posted in the app or on the website.
